Protein Reference Dataset


Protein Sequence Data

Orthology inference is most often based on molecular protein sequences. For a comparison of different orthology prediction methods, a common set of sequences must be established. Therefore, only identical proteins are mapped to each other.
To make comparisons of method easier, the orthology research community has agreed in 2009 to established a common QfO reference proteome dataset. You can choose whether your orthology predictions should be assessed on the base of this reference dataset or on the proteome set of OMA.
Note: We do no longer support the comparison against the OMA dataset

Ortholog Relation Data


Ortholog Relation Data

Our tests assess orthology on the bases of protein pairs. Therefore, we ask our users to upload their prediction in a format from which we can extract pairwise relations in an unambiguous manner: We support
  • simple simple text file with two tab-separated columns of orthologous protein represented by their ids
  • orthoxml v0.3 , which allows for nested orthologGroups and paralogGroups.
For both formats, we expect you to submit your predictions in a single file. This file might also be compressed by gzip or bzip2. In that case, it needs to have the proper filenameextention (.gz or .bz2).
